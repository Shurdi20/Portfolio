💪 FitBuddy

Personal trainer booking app — PHP · MySQL · Full-stack

Bekijk live app → GitHub →

Over het project

FitBuddy is een full-stack webapplicatie waarmee personal trainers eenvoudig hun agenda beheren en klanten zelfstandig afspraken kunnen inplannen. Het systeem werkt met drie rollen — klant, trainer en admin — elk met eigen rechten en een eigen dashboard. De app is gebouwd met PHP en MySQL zonder frameworks, met een mobile-first design.

Technologieën

PHP MySQL HTML & CSS JavaScript PDO (prepared statements) Bcrypt (wachtwoordbeveiliging) Sessie-authenticatie Mobile-first design

Functionaliteiten per rol

👤 Klant

  • Registreren & inloggen
  • Trainer selecteren
  • Datum kiezen
  • Tijdslot boeken
  • Afspraken annuleren
  • Geschiedenis bekijken

🏋️ Trainer

  • Eigen dashboard
  • Komende afspraken
  • Klantgegevens inzien
  • Statistieken
  • Beschikbaarheid bekijken

⚙️ Admin

  • Trainers toevoegen/bewerken
  • Beschikbaarheid instellen
  • Alle afspraken beheren
  • Klantenlijst
  • Rollen toekennen

Technische highlights

Rolgebaseerde toegang

Drie rollen met gescheiden routes en rechten via PHP-sessies.

Slimme tijdsloten

Automatisch gegenereerde 1-uurs slots op basis van beschikbaarheid minus bestaande boekingen.

SQL-injectie bescherming

Alle database-queries via PDO prepared statements.

Veilige wachtwoorden

Bcrypt hashing via PHP's password_hash() — industrie-standaard.

Foto upload

Trainers kunnen een profielfoto uploaden met validatie op bestandstype.

Mobile-first

Volledig responsive design dat werkt op elk schermformaat.

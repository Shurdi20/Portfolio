Over het project

De TikTok Auto Generator is een volledig geautomatiseerde AI-pipeline waarmee bedrijven zonder handmatige tussenkomst professionele short-form video's kunnen produceren. Het systeem genereert automatisch een pakkend verhaal via Claude AI, maakt daar een realistische voiceover van via ElevenLabs, downloadt een geschikte achtergrondvideo van Pexels, en combineert alles tot een TikTok-klare 9:16 MP4 — inclusief een webdashboard waarmee je met één klik een nieuwe video genereert.


Dit project is gebouwd om bedrijven te helpen hun contentproductie te versnellen en te optimaliseren. In plaats van uren te besteden aan het maken van één video, kan een medewerker op één knop drukken en binnen enkele minuten een complete, publiceerklare video downloaden.

Hoe werkt de pipeline?

1
🤖 Verhaal genereren

Claude AI genereert een pakkend verhaal van 130–160 woorden met een sterke openingszin, opbouwende spanning en een schokkeinde conclusie. Het thema wordt willekeurig gekozen uit 8 categorieën.

Claude Sonnet API
2
🎙 Voiceover maken

ElevenLabs zet de tekst om naar een dramatische, natuurlijk klinkende Engelse voiceover via het eleven_turbo_v2_5 model met geoptimaliseerde steminstellingen.

ElevenLabs API
3
🎮 Achtergrondvideo ophalen

De Pexels API downloadt automatisch rechtenvrije actievideo's in portretformaat. Clips worden lokaal opgeslagen en hergebruikt zodat er niet elke keer gedownload hoeft te worden.

Pexels API
4
🎬 Video samenstellen

ffmpeg combineert de achtergrondvideo en voiceover tot een TikTok-klare 9:16 (1080×1920) MP4. Kortere clips worden automatisch herhaald als de audio langer is.

ffmpeg · H.264 · AAC
5
🖥 Webdashboard

Een lokale Python server serveert een live dashboard. Één knop start de volledige pipeline, met een live log, voortgangsbalk en directe downloadknop voor de gegenereerde video.

Python HTTPServer · Vanilla JS · Threading

Technologieën

Python Claude API (Anthropic) ElevenLabs TTS Pexels API ffmpeg REST APIs Python HTTPServer Threading JSON prompting HTML · CSS · JavaScript

Technische highlights

API chaining

Drie externe APIs worden automatisch na elkaar aangeroepen. De output van de ene stap is de input van de volgende.

Automatisch downloaden

De tool detecteert of er genoeg clips aanwezig zijn en downloadt er automatisch meer van Pexels wanneer dat nodig is.

9:16 videoformaat

ffmpeg schaalt en snijdt elke video automatisch bij naar het TikTok-formaat van 1080×1920 pixels.

Live dashboard

Python draait een lokale webserver met een realtime log en voortgangsbalk die de pipeline stap voor stap bijhoudt.

Foutafhandeling

Elke API-aanroep heeft foutafhandeling zodat de pipeline duidelijke foutmeldingen geeft in plaats van stil te crashen.

Metadata opslaan

Elke video krijgt automatisch een JSON-bestand met het verhaal, de hook en een kant-en-klare TikTok-caption.

Waarom dit project?

Dit project heb ik gebouwd omdat ik zag dat veel bedrijven dagelijks tijd kwijt zijn aan het maken van social media content. Een video koste vroeger uren. Met deze tool druk je op één knop en heb je binnen een paar minuten een complete, publiceerklare video.


Het idee is dat bedrijven hun contentprocessen kunnen automatiseren en versnellen met behulp van AI. In plaats van een medewerker uren te laten besteden aan videoproductie, doet de tool dat volledig zelfstandig. Van verhaal genereren tot de uiteindelijke video.


Dit past precies in wat ik wil bereiken. Slimme AI-tools bouwen die echte bedrijfsproblemen oplossen en processen sneller en goedkoper maken.
